• What is the definition of server port?
  For a computer connected to a network with an IP address, a port is a communication endpoint. Ports are designated by numbers, 
  and below 1024 each port is associated by default with a specific protocol.	

• What is server port range?
  Port numbers range from 0 to 65535.
